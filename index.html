⚙️ AI 设置

选择大模型厂商，填入 API Key 即可。设置只保存在本设备。

⚠️ 关于国产模型（Kimi/通义/智谱）：它们的 API 默认不允许浏览器跨域直连（CORS），在本页面直接调用会失败。常见方案：
1️⃣ 让家里懂开发的人搭个简单代理（Cloudflare Worker、Vercel 20 行代码就能转发）；
2️⃣ 把"API 接入地址"改成你自己的代理 URL；
3️⃣ 或者直接用 OpenAI（GPT-4o 原生支持浏览器跨域）。

🔑 Kimi 用户请注意：Kimi-Coding 专用 Key 主要给代码补全工具用，通常不支持图像输入。拍照出题需要在 platform.moonshot.cn 开通 moonshot-v1-8k-vision-preview 视觉模型的 Key。
🔊 语音朗读（可选·强烈推荐）：配上服务端 TTS 后，题目/提示会用真人感童声念出。推荐 火山引擎（200 万字符/月免费，儿童音色一流）。
部署 proxy.js 后，把下方地址改成：http://<proxy 机器 IP>:8787/tts。留空则用浏览器自带 TTS（音质较差）。
